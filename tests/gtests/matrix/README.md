# Matrix gtests

This directory is generated by `tests/tools/gen_matrix`.

- Pairwise runs are emitted under `tests/gtests/matrix/pairwise/`.
- Expanded nightly coverage runs are emitted under `tests/gtests/matrix/full/`.

Generation deduplicates combinations that collapse to the same invocation and then
keeps a capped, diverse subset so the corpus exercises distinct behavior instead of
repeating equivalent permutations.

Generate:

```
./tests/tools/gen_matrix --mode pairwise
./tests/tools/gen_matrix --mode full
```
